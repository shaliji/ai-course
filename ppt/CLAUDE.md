# PPT 目录说明

## 结构

```
PPT/
├── CLAUDE.md
├── images/
├── video/
└── index.html
```

## 职责

- `index.html`：单文件网页 PPT，承载页面结构、样式、WebGL 背景与翻页逻辑。
- `images/`：存放 PPT 图片资源，文件名使用 `{页号}-{语义}.{ext}`，例如 `01-cover.jpg`。
- `video/`：存放 PPT 视频资源，页面只用相对路径引用，保证目录可整体迁移。
- `CLAUDE.md`：记录本目录的组织边界，避免交付物散落。

## 决策

本目录只服务“养龙虾课程”的 AI PPT 产物。页面使用 `guizang-ppt-skill` 的单 HTML 方案，主题采用“靛蓝瓷”，保持科技、理性、冷静的视觉基调。
